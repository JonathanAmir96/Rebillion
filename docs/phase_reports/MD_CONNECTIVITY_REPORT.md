## What the tool does

`tools/md_graph.py` walks every markdown file in the tree (117 at this audit), extracts every
cross-doc reference (markdown `[..](..)` links **and** inline `path/FILE.md` / bare-`FILE.md`
mentions — the tree's docs reference each other mostly by inline path, not link syntax),
resolves each to a real file, and builds two graphs:

- **Undirected** — an edge A—B whenever either file references the other. BFS over this graph
  answers "is the corpus one connected body of docs?"
- **Directed** — A→B when A references B. Reachability from the entry points (`README.md`,
  `CLAUDE.md`, `memory.md`) answers the stronger question "can a reader actually *navigate to*
  every doc by following links?"

Run it: `python3 tools/md_graph.py` (add `--json out.json` / `--dot out.dot` for the raw
graph or a Graphviz render). Caveat the 2026-07-24 audit confirmed: the CLI's printed
"Unreferenced" check is an in/out-degree scan, **weaker** than the true entry-point BFS this
report's methodology requires — a file cluster that links to itself (e.g. `memory/*.md`) shows
0 unreferenced while still being unreachable top-down. Use the `--json` dump + a BFS over its
`directed` adjacency for the real answer (or see the Open Question below).

## Notes for future sessions

- Re-run `tools/md_graph.py` after any wave that adds docs (especially parallel-session merges
  — that is exactly how F/G/H, and later the Memory Bank, slipped in undiscoverable). A
  self-linking cluster will pass the CLI's degree check while still failing top-down BFS — use
  the `--json` dump for the real reachability answer.
- The tool is intentionally reference-tolerant (inline paths count as edges) to match how this
  tree actually cross-links. It does **not** validate that a referenced anchor (`#section`)
  exists — that is out of scope; `tools/validate.py` owns content-level reference checks.

## Open Questions

- Should a lightweight `md_graph.py --check` mode (exit 1 on any directed-unreachable file) be

  in the batch pipeline. **Sharpened by the 2026-07-24 md audit:** if built, `--check` must do
  the true entry-point BFS, not the degree scan — the `memory/` cluster proved the degree scan
  can read 0 while five files are unreachable.
